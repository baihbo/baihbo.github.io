1. 环境搭建

1.1 系统

系统：win10

1.2 安装软件

1.2.1 node.js

  • 下载地址：https://nodejs.org/zh-cn/download/
  • 下载完成后检查npm是否安装成功
  • win+r 唤出窗口后键入 cmd 打开命令行键入 npm -v 检查版本，有版本号即为安装成功，检查过程可能较慢，请耐心等待

1.2.2 git

(Visual Studio Code已内置，可以考虑使用其内置git)

  • 下载地址https://git-scm.com/downloads

  • 连接 Github

    右键打开 Git Bash Here ，设置用户名和邮箱

    创建 SSH 密匙，记下保存位置

    用记事本从保存位置打开公钥 id_rsa.pub 并复制里边的内容

    登陆 GitHub ，进入 Settings 页面，选择左边栏的 SSH and GPG keys，点击 New SSH key

    Title 取个名字，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 Add SSH key 完成添加

    打开 Git Bash，输入 ssh -T git@github.com 出现 “Are you sure……”，输入 yes 回车确认

    出现 You've successfully ... 即成功

  • 在 Github 中创建库 用户名.github.io ，选择 “Initialize this repository with a README”
